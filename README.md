# 🏏 IPL Analytics Dashboard (2008–2025)

## Dashboard Preview

![IPL Dashboard](IPL_Dashboard.png)

## Project Overview

This Power BI dashboard provides comprehensive analysis of IPL seasons from 2008 to 2025. It offers insights into team performance, player statistics, tournament outcomes, and season-wise trends.

## Features

- Season-wise IPL Analysis
- Orange Cap & Purple Cap Statistics
- Team Performance Analysis
- Champions & Runner-Up Tracking
- Interactive Filters and Slicers
- Points Table Analysis
- KPI Cards and DAX Measures

## Tools Used

- Power BI
- DAX
- Power Query
- Excel
- Data Modeling

## Key Insights

- Analyze team standings and points table.
- Track Orange Cap and Purple Cap winners.
- Compare player performance across seasons.
- Explore match and venue statistics.
